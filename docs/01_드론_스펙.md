# 🚁 01. 모선 드론 "그물매(Geulmae)" 하드웨어 풀스펙

> 콘셉트: **"적 드론보다 ㅈㄴ 커야 한다."** 작은 놈이 큰 놈을 잡는 건 영화에서나 되는 거고, 현실에선 큰 놈이 작은 놈 잡음.

---

## 0. 설계 철학 (Design Philosophy)

1. **체급으로 압도한다.** 적 FPV가 1kg이면 우리는 그물 + 페이로드 합쳐서 그걸 들고도 날 수 있어야 함.
2. **속도 ≥ 적.** 못 따라잡으면 못 잡는다. 너무 당연.
3. **공중 정지(hover) 안정성.** 그물 조준할 때 흔들리면 빗나감.
4. **포획 후에도 비행 가능.** 추가 무게(잡은 드론 + 그물) 매달고도 귀환해야 함.
5. **모듈화.** 그물런처는 카트리지처럼 교체. (포획물은 별도 케이지 없이 그물째 운반)

## 1. 기체 형상 (Airframe)

| 항목 | 사양 |
|------|------|
| 형식 | **옥타콥터 (X8, 동축반전 4암)** — 추력 여유 + 1개 모터 고장 시에도 비행 |
| 대각 휠베이스 | **1,150 mm** (적 FPV 대비 ㅈㄴ 큼) |
| 프레임 | 카본파이버 모노코크 + 알루미늄 7075 조인트 |
| 자체 중량 | **6.8 kg** (페이로드 제외) |
| 최대 이륙중량(MTOW) | **14 kg** |
| 페이로드 여유 | **~5 kg** (그물런처 x2 + 포획 그물 + 잡은 드론 포함) |
| 방수/방진 | IP54 (비 좀 와도 출근함) |
| 색상 | 무광 다크그레이 (하늘에서 안 보이게) / 옵션: 태양 역광용 저반사 코팅 |

## 2. 추진계 (Propulsion)

| 항목 | 사양 |
|------|------|
| 모터 | 8 x 브러시리스 외전형, **KV 280** (저KV·고토크, 대형프롭용) |
| 프로펠러 | 18인치 카본 폴딩 프롭 (동축반전) |
| ESC | 8 x 80A, BLHeli_32 / 텔레메트리 지원 |
| 최대 추력 | 합계 약 **32 kgf** (추력대중량비 ≈ 2.3:1 → 기동성 확보) |
| 모터 페일세이프 | 1개 정지 시 나머지 7개로 안정 비행, 2개 정지 시 비상착륙 |

> 추력대중량비 2.3:1 = "잡은 드론 매달고도 도망 안 가고 따라잡을 수 있는" 숫자.

## 3. 비행 성능 (Flight Performance)

| 항목 | 사양 |
|------|------|
| 최고 수평속도 | **120 km/h** (적 FPV 추격용 스프린트 모드) |
| 순항 속도 | 60 km/h |
| 최대 상승률 | 12 m/s |
| 실용 상승한도 | 5,000 m (고지대 작전 가능) |
| 호버링 정밀도 | ±0.1 m (RTK GPS + 비전 측위 시) |
| 최대 풍속 견딤 | 15 m/s (강풍에도 조준 유지) |
| 체공시간 (무장착) | **42분** |
| 체공시간 (그물런처+포획 그물 매달고) | **~25분** |
| 작전 반경 | 기지 기준 반경 8 km |

---

## 5. 컴퓨트 (Onboard Compute)

| 항목 | 사양 |
|------|------|
| 메인 AI 컴퓨트 | **NVIDIA Jetson Orin (AGX급, ~275 TOPS)** — HawkEye-VLM 온디바이스 추론 |
| 비행컨트롤러 | Pixhawk 6X급 (PX4/ArduPilot), 트리플 IMU 이중화 |
| 보조 MCU | 그물런처/윈치/릴리스 액추에이터 실시간 제어용 STM32 |
| 저장장치 | 1TB NVMe (블랙박스 로깅 + 노획영상 저장) |
| OS | 비행: NuttX(RTOS) / AI: Ubuntu + ROS 2 |

> 핵심: 통신 끊겨도 **온디바이스에서 판별·포획 결심까지 자율**로 한다. 적이 재밍해도 그물매는 일함.

## 8. 안전장치 (Safety)

- **지오펜스:** 인구밀집지역 자동 회피 (투하 금지구역 DB 내장)
- **낙하산:** MTOW 대응 자동전개 낙하산 (모선 추락 시 지상피해 최소화)
- **킬스위치:** 지상국 원격 + 자율판단 비상모드
- **포획물 격리:** 포획 그물은 폭발/화재 시 즉시 그물째 분리투하(jettison) 가능
- **IFF 연동:** 아군 드론 식별코드 수신 시 포획 자동 잠금
