# 🪤 04. 추가 부품 — 그물런처 · 둥지 · 투하 메커니즘

> 그물매를 "대드론 사냥꾼"으로 만드는 진짜 핵심 장비들. 드론 본체는 그냥 **그물을 운반하는 택배기사**고, 진짜 일꾼은 얘네다.

---

## 1. 그물런처 (NetLauncher) — 핵심 무장

> "5만원짜리 드론한테 미사일 쏘지 말고, **5천원짜리 그물** 던지자."

| 항목 | 사양 |
|------|------|
| 형식 | 압축가스(CO₂/압축공기) 구동 다발(多錘) 그물 발사기 |
| 발사 방식 | 4~6개 추(weight)가 그물 모서리를 끌고 부채꼴로 전개 |
| 그물 재질 | **케블라/UHMWPE(다이니마) 다중가닥** — 가볍고 ㅈㄴ 질김, 안 끊김 |
| 그물 크기 | 전개 시 약 **3 x 3 m** (적 드론 통째로 덮음) |
| 유효 사거리 | **8~15 m** (근접 정밀포획) |
| 전개 시간 | 발사 후 0.3초 내 완전 전개 |
| 특수 기능 | **자기수축(self-cinching):** 표적 닿으면 그물 입구가 조여져 프로펠러 결속 |
| 장탄 | 카트리지식 1발 (그물런처당), MAX형은 재장전 매거진 |
| 재장전 | 기지 복귀 후 카트리지 교체 (30초) / 옵션: 공중 재장전 매거진 |

```
   발사 전 (접힘)          발사 직후 (전개)
   ┌─────────┐            ●        ●
   │ ▣ 그물  │   ──▶       ╲  ╱╲  ╱
   │ 카트리지 │            ╲╱  ╲╱   ← 추 4개가 끌고 부채꼴로 펼쳐짐
   └─────────┘             ╲网网网╱
   (CO₂ 챔버)               표적 덮음 → 자기수축 → 프롭 엉킴
```

### 1.1 왜 압축가스냐 (화약 안 씀)
- 화약식은 반동·소음·법규 이슈. 압축가스는 **재충전 가능 + 조용 + 합법적**.
- 반동이 작아서 그물매 자세가 덜 흔들림 → 연속 포획 유리.

### 1.2 자기수축(Self-Cinching) 메커니즘
- 그물 가장자리에 슬립 코드(draw cord) → 추의 관성으로 표적 감싸는 순간 입구가 졸려듦.
- 효과: 적 드론이 빠져나가려고 발버둥쳐도 프로펠러가 그물에 더 엉킴. **발버둥칠수록 더 갇힘.** (개미지옥 원리 ㅋ)

## 2. 둥지 (Nest) — 포획물 격리 케이지

> 잡은 드론을 담아 **기지까지 안전하게 데려오는** 캐리어. 동시에 "혹시 터지면" 대비한 격리실.

| 항목 | 사양 |
|------|------|
| 형식 | 접이식 메쉬 케이지 (포획 시 전개, 비행 시 접힘) |
| 용량 | 소형~중형 드론 1기 (MAX형은 2기) |
| 재질 | 카본 프레임 + 케블라 메쉬 |
| 격리 기능 | 포획물 고정 + **충격/파편 일부 흡수 라이너** |
| 비상 분리 | **긴급 투하(Jettison):** 둥지 안에서 발화/기폭 징후 시 0.2초 내 통째 분리낙하 |
| 센서 | 둥지 내부 IR/가스센서 (포획물 발열·연기 감시) |

### 2.1 둥지의 이중생활
- **양품일 때:** 잡은 드론을 곱게 담아 기지로 모셔감 (분석용 VIP 손님)
- **위험할 때:** 애초에 둥지에 안 넣음. 그물째 매달고 가서 버림 (3번 참조)
- **넣었는데 갑자기 ㅈ될 때:** 둥지 내부 센서가 발열/연기 감지 → 즉시 jettison

## 3. 회수/투하 메커니즘 (Retrieve / Drop)

> TaloNet의 시그니처. **"가져오거나, 버리거나."** 둘 중 하나.

### 3.1 회수 모드 (CAPTURE_AND_RETURN)
1. 그물런처 발사 → 표적 포획
2. 윈치(winch)로 그물+포획물을 둥지로 끌어올림
3. 둥지 폐쇄 → 격리 → 기지로 귀환
4. 기지에서 인도 → 정보분석팀 행

```
포획 ─▶ 윈치 회수 ─▶ 둥지 격리 ─▶ 🏠 기지 귀환 ─▶ 분석
```

### 3.2 투하 모드 (CAPTURE_AND_DROP) — "이거 가져오면 ㅈ됨"
1. 그물런처 발사 → 표적 포획 (단, **둥지에 안 넣음**, 그물째 하방 매달기)
2. **무인지대 탐색:** 지오펜스 DB + 실시간 IR/EO로 "사람·차량·건물 없는 곳" 확인
3. 안전지대 상공 도착 → 안전고도 확보
4. **그물 분리(release):** 전자식 릴리스로 그물째 투하
5. 적 드론, 그물에 엉킨 채 무인지대 낙하 → (자폭이면) 거기서 펑

```
포획(매달기) ─▶ 무인지대 탐색 ─▶ 안전고도 ─▶ 🗑️ 그물 분리 투하 ─▶ 안전이격
```

### 3.3 무인지대 판정 (Safe-Drop Zone Logic)
- 내장 지오펜스: 도심/도로/시설 = 투하 금지구역(빨강)
- 실시간 센서: EO/IR로 투하지점에 사람·동물·차량 없는지 최종 확인
- 후보지 우선순위: 들판 > 하천부지 > 빈 공터 > (최후) 자기 기지 안전구역
- 투하 후 모선은 즉시 수평 이격 (파편 회피)

## 4. 윈치 & 릴리스 시스템 (Winch & Release)

| 부품 | 역할 |
|------|------|
| 전동 윈치 | 포획물을 둥지로 끌어올림 (회수 모드) |
| 전자식 릴리스 후크 | 그물째 즉시 분리 (투하/비상 모드) |
| 텐션 센서 | 매달린 무게/요동 측정 → 비행제어에 피드백(흔들림 보정) |
| 댐퍼 | 매달린 포획물 진자운동 억제 |

> 매달린 물체가 흔들리면 모선이 같이 춤춘다. 그래서 텐션센서+댐퍼로 "흔들리는 추 들고 안정비행" 문제를 해결.

## 6. 소모품 BOM (Bill of Materials, 소모품 위주)

```yaml
consumables:
  net_cartridge:        # 1회용. 사업의 핵심 수익원 ㅋㅋ
    material: Kevlar/UHMWPE multi-strand
    deployed_size_m: 3x3
    weights: 4-6 (corner)
    self_cinching: true
  co2_canister:         # 그물런처 추진제
    type: rechargeable compressed gas
  nest_liner:           # 격리 라이너 (오염/손상 시 교체)
    type: impact-absorbing kevlar mesh
reusable:
  net_launcher: gas-powered, cartridge-reload
  nest_cage: foldable carbon+kevlar
  winch: electric, tension-sensed
  release_hook: electronic quick-release
```

## 7. 부품-기능 매핑 (한눈에)

| 적 드론 상황 | 동원 부품 | 결과 |
|--------------|-----------|------|
| 흔한 정찰 드론 | 그물런처 → 윈치 → 둥지 | 기지 회수 ✅ |
| 신형/고가치 드론 | 그물런처 → 윈치 → 둥지 | 회수 후 분석 🔬 |
| 자폭 의심 드론 | 그물런처 → 매달기 → 릴리스 | 무인지대 투하 🗑️ |
| 둥지 안에서 발열 | 둥지 IR센서 → jettison | 긴급 분리낙하 🚨 |
| 군집 다수 | 그물런처 연속발사(MAX형) | 우선순위순 처리 |

> ⚠️ 본 문서는 **기획 단계 부품 설계**입니다. 실제 시제품/SCAD/시험은 Phase 1~4에서 진행됩니다.
