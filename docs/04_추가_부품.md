# 🪤 04. 추가 부품 — 그물런처 · 그물운반 · 투하 메커니즘

> Peregrine를 "대드론 사냥꾼"으로 만드는 진짜 핵심 장비들. 드론 본체는 그냥 **그물을 운반하는 택배기사**고, 진짜 일꾼은 얘네다.

## 1. 그물런처 (NetLauncher) — 핵심 무장

> "5만원짜리 드론한테 미사일 쏘지 말고, **5천원짜리 그물** 던지자."

| 항목 | 사양 |
|------|------|
| 형식 | 압축가스(CO₂/압축공기) 구동 다발(多錘) 그물 발사기 |
| 발사 방식 | 4~6개 추(weight)가 그물 모서리를 끌고 부채꼴로 전개 |
| 그물 재질 | **케블라/UHMWPE(다이니마) 다중가닥** — 가볍고 ㅈㄴ 질김, 안 끊김 |
| 그물 크기 | 전개 시 약 **3 x 3 m** (적 드론 통째로 덮음) |
| 유효 사거리 | **8~15 m** (근접 정밀포획) |
| 전개 시간 | 발사 후 0.3초 내 완전 전개 |
| 특수 기능 | **자기수축(self-cinching):** 표적 닿으면 그물 입구가 조여져 프로펠러 결속 |
| 장탄 | 카트리지식 1발 (그물런처당), MAX형은 재장전 매거진 |
| 재장전 | 기지 복귀 후 카트리지 교체 (30초) / 옵션: 공중 재장전 매거진 |

```
   발사 전 (접힘)          발사 직후 (전개)
   ┌─────────┐            ●        ●
   │ ▣ 그물  │   ──▶       ╲  ╱╲  ╱
   │ 카트리지 │            ╲╱  ╲╱   ← 추 4개가 끌고 부채꼴로 펼쳐짐
   └─────────┘             ╲网网网╱
   (CO₂ 챔버)               표적 덮음 → 자기수축 → 프롭 엉킴
```

### 1.1 왜 압축가스냐 (화약 안 씀)
- 화약식은 반동·소음·법규 이슈. 압축가스는 **재충전 가능 + 조용 + 합법적**.
- 반동이 작아서 Peregrine 자세가 덜 흔들림 → 연속 포획 유리.

### 1.2 자기수축(Self-Cinching) 메커니즘
- 그물 가장자리에 슬립 코드(draw cord) → 추의 관성으로 표적 감싸는 순간 입구가 졸려듦.
- 효과: 적 드론이 빠져나가려고 발버둥쳐도 프로펠러가 그물에 더 엉킴. **발버둥칠수록 더 갇힘.** (개미지옥 원리 ㅋ)

## ⚔️ 1.5 두 개의 그물 — CRADLE(생포) vs TRAWLER(폐기) 〔교전수칙 핵심〕

> 적 전쟁드론은 크게 두 종류다. **정찰 드론**(카메라·통신 = 정보 가치, 살려서 취조)과 **자폭/공격 드론**(폭탄 그 자체, 만지면 우리가 죽음). 한 가지 그물로 둘 다 상대하면 안 된다. 그래서 그물을 둘로 나눈다 — **하나의 조준 터렛, 두 개의 탄종.**

| | 🕊️ **CRADLE** (요람 그물) | 💥 **TRAWLER** (저인망 그물) |
|---|---|---|
| 대상 | 정찰 드론 (생포) | 자폭/공격 드론 (폐기) |
| 발사 | `RELAY 0` (기존 CO₂ 런처) | **`RELAY 1` 대형 표준오프 런처** |
| 그물 | 소형 자기수축 그물 (≈3×3 m) | **대형 광역 그물 (≈6×6 m), 고밀도 메시** |
| 포박 | `SERVO 11` cinch로 입구 졸라 무손상 결속 | 중량추로 통째 엉킴 — 로터 완전 정지 |
| 분리 | `SERVO 12` 윈치로 벨리에 끌어올림 | **`RELAY 2` cord-cutter(소선/블레이드)로 테더 절단** |
| 거리 | 근접 (접촉 OK — 양품이니까) | **표준오프 = 절대 접촉 금지** |
| 결말 | 그물째 곱게 모셔가 분석(생포→취조) | 줄 끊고 드론째 무인지대 투기 (자폭하든 말든 우리와 분리) |

### 왜 자폭 드론엔 별도 그물이 필요한가
기존 "투하 모드"는 **표적에 근접해서 cinch로 입을 졸라** 매단 뒤 버리는 방식이었다. 문제: **폭탄한테 뽀뽀하러 가는 것.** cinch하려고 접근하는 순간 기폭되면 모선이 같이 ㅈ된다.

→ **TRAWLER는 접근 자체를 안 한다.** 안전거리에서 큰 그물을 던져 **로터를 엉키게 해 추락만 막고**, 곧바로 **cord-cutter로 테더를 끊어** 드론을 그물째 떨궈버린다. 모선은 단 한 순간도 폭탄과 물리적으로 연결되지 않는다. cinch도, 윈치 회수도 없다 — **잡지 않고 떨군다.**

> 조종 UI(`gcs/app.py`)는 현재 장전된 그물을 레티클 색/크기로 구분: **CRADLE=초록 소형링**, **TRAWLER=빨강 대형 이중링.** 키 `M`으로 전환, `SPACE`로 발사, `X`로 cord-cut. 인터락상 cinch는 CRADLE 전용, cord-cut은 TRAWLER 전용이라 헷갈려 잘못 누를 수 없다. (채널 정의: `gcs/payload_map.py`)

## 2. 그물 운반/수축 (Net Carry & Cinch) — **CRADLE 전용**

> ⚠️ 아래 운반/수축/윈치는 **CRADLE(정찰 생포)** 경로다. TRAWLER(자폭)는 운반하지 않고 cord-cut으로 즉시 분리하므로 해당 없음.


> 잡은 드론을 **그물 그대로** 기지까지 데려온다. 별도 수납 케이지(둥지)는 **없앴다.**
> 그물이 곧 캐리어다 — 포획 → 입구 수축(cinch) → 벨리 밑에 매단 채 운반.

| 항목 | 사양 |
|------|------|
| 운반 방식 | 포획 그물을 **하방 매달기** 상태로 운반 (케이지 X) |
| 입구 고정 | **드로스트링 수축(cinch) + 윈치 장력**으로 그물 입구를 졸라 포획물 이탈 방지 |
| 안정화 | 윈치로 벨리 근처까지 끌어올려 진자운동 최소화 + 텐션센서/댐퍼 |
| 위험물 감시 | 그물 어셈블리 IR/가스 센서로 포획물 발열·연기 감시 |
| 비상 분리 | **긴급 투하(Jettison):** 발화/기폭 징후 시 0.2초 내 그물째 전자식 릴리스로 분리낙하 |

### 2.1 그물 운반의 이중생활
- **양품일 때:** 그물 입구 수축 → 벨리 밑에 곱게 매달고 기지로 모셔감 (분석용 VIP 손님)
- **위험할 때:** 수축은 하되 끌어올리지 않고 그물째 길게 매달고 가서 버림 (3번 참조)
- **갑자기 ㅈ될 때:** 그물 IR/가스 센서가 발열/연기 감지 → 즉시 그물째 jettison

## 3. 회수/투하 메커니즘 (Retrieve / Drop)

> TaloNet의 시그니처. **"가져오거나, 버리거나."** 둘 중 하나.

### 3.1 회수 모드 (CRADLE / CAPTURE_AND_RETURN) — 정찰 드론 생포
1. **CRADLE** 그물런처 발사(`RELAY 0`) → 표적 포획
2. **그물 입구 수축(cinch, `SERVO 11`)** → 포획물 이탈 방지
3. 윈치(winch, `SERVO 12`)로 그물 포켓을 벨리 근처까지 끌어올려 안정화
4. 그물째 매단 상태로 기지 귀환 → 인도 → 정보분석팀(포렌식) 행

```
CRADLE 발사 ─▶ 입구 수축 ─▶ 윈치로 벨리 근처 안정화 ─▶ 🏠 기지 귀환(그물째) ─▶ 취조
```

### 3.2 무력화 모드 (TRAWLER / NEUTRALIZE_AND_JETTISON) — "이거 만지면 ㅈ됨"
> 핵심: **접근하지 않는다.** cinch도 윈치도 없다. 표준오프(원거리)에서 던지고, 끊고, 떨군다.

1. **안전거리 유지** — 자폭 반경 밖에서 조준(`M`으로 TRAWLER 선택, 레티클 빨강 대형)
2. **TRAWLER 대형 그물 발사(`RELAY 1`)** → 광역 그물이 표적을 통째로 덮어 **로터 결속 → 추진력 상실 → 추락 저지**
3. **무인지대 확인** — 지오펜스 DB + 실시간 IR/EO로 낙하지점에 사람·차량·건물 없는지 확인
4. **cord-cutter 작동(`RELAY 2`, 키 `X`)** — 소선(니크롬)/블레이드로 테더 절단 → 그물+드론이 모선과 **완전 분리**
5. 적 드론, 그물에 엉킨 채 무인지대 자유낙하 → (자폭이면) **모선과 멀리 떨어진 곳에서** 펑

```
표준오프 조준 ─▶ TRAWLER 발사(대형) ─▶ 로터 엉킴/추락저지 ─▶ 무인지대 확인 ─▶ ✂️ cord-cut 분리 ─▶ 🗑️ 그물째 투기
```

> 모선이 폭탄과 물리적으로 연결되는 시간 = **0초.** 이게 기존 투하모드와의 결정적 차이다.

### 3.3 무인지대 판정 (Safe-Drop Zone Logic)
- 내장 지오펜스: 도심/도로/시설 = 투하 금지구역(빨강)
- 실시간 센서: EO/IR로 투하지점에 사람·동물·차량 없는지 최종 확인
- 후보지 우선순위: 들판 > 하천부지 > 빈 공터 > (최후) 자기 기지 안전구역
- 투하 후 모선은 즉시 수평 이격 (파편 회피)

## 4. 윈치 & 릴리스 & cord-cutter (Winch / Release / Cord-Cutter)

| 부품 | 그물 | 역할 |
|------|------|------|
| 전동 윈치 | CRADLE | 그물 포켓을 벨리 근처로 끌어올려 안정화 (회수 모드) |
| 전자식 릴리스 후크 | CRADLE | 회수 그물 비상 분리 (비상 투하) |
| **cord-cutter (소선/블레이드)** | **TRAWLER** | **`RELAY 2` — 표준오프 그물의 테더를 절단해 자폭 드론을 그물째 투기.** 니크롬 소선 통전(burn-wire) 또는 솔레노이드 블레이드. fire-and-forget. |
| 텐션 센서 | CRADLE | 매달린 무게/요동 측정 → 비행제어에 피드백(흔들림 보정) |
| 댐퍼 | CRADLE | 매달린 포획물 진자운동 억제 |

> 매달린 물체가 흔들리면 모선이 같이 춤춘다. 그래서 텐션센서+댐퍼로 "흔들리는 추 들고 안정비행" 문제를 해결.

## 6. 소모품 BOM (Bill of Materials, 소모품 위주)

```yaml
consumables:
  cradle_net_cartridge:   # CRADLE 소형 그물. 1회용, 사업의 핵심 수익원 ㅋㅋ
    material: Kevlar/UHMWPE multi-strand
    deployed_size_m: 3x3
    weights: 4-6 (corner)
    self_cinching: true
  trawler_net_cartridge:  # TRAWLER 대형 표준오프 그물. 1회용
    material: Kevlar/UHMWPE multi-strand (high-density mesh)
    deployed_size_m: 6x6
    weights: 6-8 (heavier, wider spread)
    self_cinching: false   # cinch 안 함 — 엉킴 + cord-cut으로 처분
  cord_cutter_element:    # TRAWLER 테더 절단용 소선(니크롬). 1회용
    type: nichrome burn-wire (or solenoid blade, reusable)
  co2_canister:           # 그물런처 추진제 (양 런처 공용 규격)
    type: rechargeable compressed gas
reusable:
  cradle_launcher: gas-powered, cartridge-reload (RELAY0)
  trawler_launcher: gas-powered, large-bore stand-off (RELAY1)
  cord_cutter: burn-wire / solenoid blade driver (RELAY2)
  winch: electric, tension-sensed (CRADLE)
  release_hook: electronic quick-release (CRADLE)
  cinch_drawstring: net mouth closure (CRADLE carry mode, SERVO11)
```

## 7. 부품-기능 매핑 (한눈에)

| 적 드론 상황 | 그물 | 동원 부품 | 결과 |
|--------------|------|-----------|------|
| 흔한 정찰 드론 | 🕊️ CRADLE | 런처(R0) → cinch(S11) → 윈치(S12) | 그물째 기지 회수 ✅ |
| 신형/고가치 정찰 | 🕊️ CRADLE | 런처(R0) → cinch(S11) → 윈치(S12) | 생포 후 취조 🔬 |
| 자폭/공격 드론 | 💥 TRAWLER | 표준오프 대형런처(R1) → cord-cut(R2) | 접촉 없이 무인지대 투기 🗑️ |
| 자폭 의심 → 확정 | 💥 TRAWLER | 거리 유지 + 대형 그물 + 즉시 cord-cut | 모선-폭탄 연결 0초 🚨 |
| 군집 다수 | 혼합 | 표적별 CRADLE/TRAWLER 선택(`M`) 연속 처리 | 우선순위순 처리 |

> ⚠️ 본 문서는 **기획 단계 부품 설계**입니다. 실제 시제품/SCAD/시험은 Phase 1~4에서 진행됩니다.
